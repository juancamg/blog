## ¿Qué es MDX?

MDX es una extensión de Markdown que te permite usar JSX en tu contenido Markdown. Esto significa que puedes importar componentes de React y usarlos directamente en tus archivos `.mdx`. Esto abre un mundo de posibilidades para crear contenido interactivo y dinámico.

## Cómo Usar MDX

Aquí hay algunos ejemplos de lo que puedes hacer con MDX:

### Importar Componentes

Puedes importar cualquier componente de React y usarlo en tu contenido MDX. Por ejemplo:

```jsx
import { Button } from "@/components/ui/button";

<Button variant="primary">Haz clic aquí</Button>
```

### Crear Tablas

MDX también te permite crear tablas fácilmente:

```md
| Encabezado 1 | Encabezado 2 |
| ------------ | ------------ |
| Fila 1       | Fila 1       |
| Fila 2       | Fila 2       |
```

### Incluir Código con Resaltado de Sintaxis

Puedes incluir bloques de código con resaltado de sintaxis:

```ts
const greet = (name: string) => {
  return `Hola, ${name}!`;
};

console.log(greet("Mundo"));
```

### Añadir Imágenes

Añadir imágenes es muy sencillo:

```md
![Descripción de la imagen](/images/blog/welcome.webp)
```

### Crear Listas

Puedes crear listas ordenadas y desordenadas:

```md
- Elemento 1
- Elemento 2
  - Sub-elemento 1
  - Sub-elemento 2

1. Primer paso
2. Segundo paso
3. Tercer paso
```

### Bloques de Citas

También puedes añadir bloques de citas:

```md
> Este es un bloque de cita.
```

### Componentes Personalizados

Puedes crear y usar tus propios componentes personalizados. Por ejemplo, un componente de alerta:

```jsx
import { Alert } from "@/components/ui/alert";

<Alert type="warning">Este es un mensaje de advertencia.</Alert>
```
